ネタバレ防止（隠す）

文字・URL・画像・ファイルを隠すことができます。

文字・URLを隠す手順​

テキスト・URLを|| ||で囲むか、/spoilerを実行すると隠せます。

画像・ファイルを隠す手順​

添付後、目のマークをクリックするか、添付したものをタップして「スポイラーとしてマークする」をタップすると隠せます。
