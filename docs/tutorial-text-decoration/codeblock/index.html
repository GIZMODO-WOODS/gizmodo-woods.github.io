コードブロック

コマンド・プログラムなどをわかりやすく書くことができます。

１行で表示する手順​

テキストを``で囲むと、インラインコードブロックとして表示できます。

複数行で表示する手順​

テキストの上下を```で囲むと、コードブロックとして表示できます。
また、上部の```の横にプログラミング言語名を入れると、ハイライトが表示されます。
