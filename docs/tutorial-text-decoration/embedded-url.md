# リンクを埋め込む

文字にリンクを自由に埋め込むことができます。

## 手順

`[テキスト](URL)`で埋め込めます。   
[リンクのプレビューを隠す](/docs/tutorial-text-decoration/hide-preview.md)などと組み合わせられます。
