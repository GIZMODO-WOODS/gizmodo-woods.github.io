文字の大きさを変える

テキストの大きさを変えることで、わかりやすい文章を作成できます。

手順​

各行の文字の前に記号をつけることで、テキストの大きさを変えられます。
小さい順に並び替えると以下のようになります。

-# こんにちは  
こんにちは  
### こんにちは  
## こんにちは  
# こんにちは
注意

不用意に使いすぎると逆に見にくくなってしまいます。
